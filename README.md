## ⚙️ Installation

### Clone the Repository

```bash
git clone https://github.com/AnushaB20/portfolio1.git
```

### Navigate to the Project

```bash
cd portfolio1
```

### Install Dependencies

```bash
npm install
```

### Run the Application

```bash
npm run dev
```

Open your browser and visit:

```
http://localhost:5173
```

---
